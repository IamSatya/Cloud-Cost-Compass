# **App Name**: CloudCost Compass

## Core Features:

- AWS Cost Tracking: Connect to AWS accounts and track costs.
- Daily Cost Dashboard: Display daily AWS costs per account with auto-refreshing data.
- Weekly Cost Dashboard: Show weekly AWS costs per account.
- Monthly Cost Dashboard: Display monthly AWS costs per account.
- Dynamic Cost Details: Automatically update cost data in real-time on the dashboards.
- Service-Wise Cost Overview: Break down AWS costs by service (e.g., EC2, S3, Lambda).

## Style Guidelines:

- Primary color: Dark blue (#3498DB) to reflect the serious, business-oriented nature of cloud management.
- Background color: Very light blue (#ECF0F1) for a clean, professional look that ensures readability.
- Accent color: Light green (#2ECC71) to highlight key data points and provide positive visual cues for cost savings or optimization.
- Body and headline font: 'Inter' for a modern, machined, neutral look, that works well for data-rich interfaces.
- Use simple, clear icons to represent AWS services and cost metrics.
- Dashboard layout: Organize data with clear sections for daily, weekly, and monthly costs. Prioritize key metrics at the top.
- Use subtle transitions and animations for auto-refreshing data updates.
